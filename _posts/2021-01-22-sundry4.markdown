> 实现一个文件在多处的调用，而且无需挪动文件位置，无需复制，好酷🕶️，之前在学习Linux知识的时候真是小看了它。

> 利用软连接来实现 ln -s

```Bash
sudo rm -rf /usr/local/cuda #删除之前生成的软链接

sudo ln -s /usr/local/cuda-10.1 /usr/local/cuda #生成新的软链接
```
