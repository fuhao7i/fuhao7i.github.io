# 1. 保存模型

```python
torch.save(model.state_dict(),"./Double.pth")
```

# 2. 加载训练好的模型

```python
# 创建模型
model=Net()
# 加载预训练模型的参数
model.load_state_dict(torch.load("./Double.pth"))
```
